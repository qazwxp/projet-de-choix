Tutoriel du HTML

Le HTML(Hyper Text Markup Language) est un langage de codage qui est souvent utilisé pour créer les sites web. Presque tous les sites web que tu vois sont faits avec le HTML. dans ce tutoriel tu vas apprendre comment faire un site web comme celui que tu es sur, en commençant avec les basiques. Pour faire les sites du HTML plus interesant, tu as aussi besoin de la CSS(Cascading Style Sheets) pour ajouter style.
