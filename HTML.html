Les Essentiels

Dans le HTML, tous les éléments vont commencer avec <(élément)> et finir avec </(élément)>. Par exemple pour faire un paragraphe, tu dois faire <p>(texte)</p> pour créer le texte.
Tou les documents HTML ont besoin de quelques choses.

  • <!DOCTYPE html> va en haut de toute la code, il dit le navigateur que ton document est fait en HTML
  • <HTML>(code)</HTML> entour tout ton code (sauf le DOCTYPE html). Tu peux aussi mettre l'attribut "lang" dans ça (<HTML lang="eng/fr etc.">) pour aider le navigateur. Ça dit la langue de ta page.
  • <head> (tête) est utilisé pour contenir l'important information de ta page comme le titre. Plus d'information sur les éléments dans le <head> ici.
  • <body>entoure les choses qui vont être montrées sur ta page, comme le texte ou les form.

Maintenant cette code montre rien, mais ça c'est les éléments essentiels que tu dois mettre dans ta page.

Les Éléments importants

  • <p> veut dire paragraphe. Il est utilisé pour faire les paragraphes.


  • <h> veut dire "header" en anglais. Il est utilisé pour faire les titres. Tu peux utiliser h1-h6, 1 est le plus important et six et le moins important


  • <a> est utilisé pour faire les attributs. Les attributs peuvent ajouter l'information sur les éléments et leur dire comment acter. Plus info sur les attributs ici.


  • <br> peut être utilisé n'importe où dans un document pour créer une brise dans la ligne ("break" en anglais). Il peut aussi créer un espace entre les éléments.


  • <ul est utilisé pour faire les listes ("unordered list" en anglais). Dans ces listes, tu dois mettre <li> ("list item" en anglais) pour faire les points. Tu peux mettre les listes dans les listes pour créer les indentations des certains points.


  • <ol est utilisé de la même façon que <ul mais il utilise les nombres au lieu des points ("ordered list" en anglais).


  • <div> est utilisé pour créer les sections divisées dans ton code. Ça peut aider avec mettre un style différent dans une section. Il est comme un contenant pour une section de ton code.


    CSS

    Le CSS ("Cascading Style Sheets" en anglais) est essentiel pour faire un bon site web. Il peut styler les éléments dans beaucoup de façon, sois c'est la couleur, font ou frontière. Voici quelques-uns des éléments de CSS qui sont souvent utilisés.

  • color (couleur en français) change le couleur d'un élément, mais ne confuse ça pas avec le couleur du arriere plan
  • background-color change la couleur du fond d'un élément.
  • font-family change le font de le texte dans un élément.
  • font-size change la grandeur du texte dans un élément. Il utilise les pixel (exemple 20px)
  • text-align change comment le texte et aligné

    Les Attributs

    Les Attributs sont les choses que tu peux ajouter aux éléments pour ajouter information plus spécifique. Tu ajoutes les éléments dans l'étiquette d'ouvrit d'un élément. Par exemple <p (attribut)="(info)">(texte)</p>
    Les exemples des attributs sont:

    • src est utilisé pour ajouter les sources a les éléments comme les images, sons, vidéos et plus. Dans ces éléments, il dit la code ou trouver l'image ou vidéo, etc. L'image que tu ajoutes dois être dans le même dossier que ton code.
    • href est utilisé pour faire les liens. Si tu le mis dans un élément, il devient un lien et si tu le cliques il te prend a une source. Tu peux aussi l'utiliser pour aller à une certaine élément dans ta page par mettre #(id du élément) au lieu du lien.
    • id est utilisé pour identifier les éléments. Ça c'est utile pour beaucoup de raisons. Tu peux ajouter le style aux éléments individuellement dans le CSS par ajouter un "#" avant l'id et ajouter les choses normalement.

    La Texte dans HTML

    Dans le HTML, tu peux utiliser le CSS pour faire beaucoup avec la texte. Comme j'ai dit, tu peux utiliser les id pour individuellement ajouter le style a la texte, voici les different choses que tu peux changer:

    Il y a aussi des different étiquettes que tu peux toujours faire avec le HTML dans les éléments de texte pour ajouter le changer dans plusieurs different facons. voici des examples:

  • <b> est uttiliser pour la texte gras ("bold" en anglais)

  • <i> est uttiliser pour la texte italique

  • <mark> est uttiliser pour la texte marqué ("marked" en anglais)

  • <del> est uttiliser pour la texte supprimé ("deleted" en anglais)

  • <ins> est uttiliser pour la texte souligner ("inserted" en anglais parce qu'il est utiliser avec la texte supprimé)

  • <sub> et <sup> sont uttilisés pour faire la texte en indice et exposant ("subscript" et "superscript" en anglais)

    Avec le CSS, tu peux changer ces étiquettes et ajouter n'importe quoi comme étiquette, par exemple utiliser une étiquette appelé "aaaaaaaaaaa", puis ajouter une section pour le dans le CSS comme un élément normale.
